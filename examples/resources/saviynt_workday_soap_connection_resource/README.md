# saviynt_workday_soap_connection_resource

Use the following operations to perform full lifecycle management of your Workday SOAP connector. They allow you to:

- **Create** a new Workday SOAP connector with the settings you need  
- **Read** (Retrieve) the connector's current configuration  
- **Update** its configuration

[See Saviynt documentation for more details](https://docs.saviyntcloud.com/bundle/WD2-v24x/page/Content/Using-Classic-Integration.htm)

- Simple example [can be found here](./resource.tf).
