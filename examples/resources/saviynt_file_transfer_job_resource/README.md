# saviynt_file_transfer_job_resource

Use the following operations to perform full lifecycle management of your File Transfer Job triggers. They allow you to:

- **Create** new File Transfer Job triggers with the settings you need  
- **Update** trigger configuration
- **Delete** triggers from Saviynt

This resource supports bulk operations, allowing you to manage multiple File Transfer Job triggers in a single resource block.

[See Saviynt documentation for more details](https://docs.saviyntcloud.com/bundle/EIC-Admin-v23x/page/Content/Chapter10-Job-Control-Panel/Adding-a-new-job.htm)

- Simple example [can be found here](./resource.tf).
